# App #

## Requirements for running the app
---
* python 3.8
* pip
* pipenv

## How do I get set up? ##
---
* Enter a virtual environment and install dependencies
```
pipenv shell
pip install -r requirements.txt
```
* run migrations for the database:
```
./manage.py makemigrations
./manage.py migrate
```

Run the server!
```
./manage runserver
```
